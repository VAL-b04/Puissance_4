# Puissance 4

Une **version améliorée** d’un projet de lycée : un **Puissance 4** interactif en **Python**, utilisant **Pygame** pour l’interface graphique.

---

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/your-username/Puissance_4.git
cd Puissance_4

# Installer les dépendances
pip install pygame
```

### Bash
```bash
# Lancer le jeu
python3 "puissance 4.py"
```

### Powershell
```bash
# Lancer le jeu
python ".\puissance 4.py"
```
