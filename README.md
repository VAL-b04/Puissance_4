# Puissance 4

Une **version améliorée** d’un projet de lycée : un **Puissance 4** interactif en **Python**, utilisant **Pygame** pour l’interface

---

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/VAL-b04/Puissance_4
cd Puissance_4

# Installer les dépendances
pip install pygame

# Lancer le jeu
python3 "puissance_4.py"
```
